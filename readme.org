* System Override Saga (SOS)

A game written in Fennel + Love2D for the  [[https://itch.io/jam/autumn-lisp-game-jam-2025][Autumn Lisp Game Jam 2025]]. Templated with the [[https://codeberg.org/alexjgriffith/min-love2d-fennel]] project, but the game idea + code is my own.

** Note on LLMs/AI:

No LLMs or generative AI were used in the creation of this game. No PRs using LLMs will be accepted. No exceptions.
